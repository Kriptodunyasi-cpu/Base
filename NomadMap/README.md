# 🌍 NomadMap: The Base Digital Nomad Hub

**NomadMap** is a community-driven map platform with on-chain verification built on the **Base Network**. Digital nomads share their experiences and earn **NCAT** rewards for contributing to the decentralized ecosystem.

---

## 🚀 About the Project
Unlike traditional map apps, NomadMap integrates the nomadic lifestyle with the Base ecosystem. Travelers "check-in" at locations like cafes, campsites, or restaurants, recording their visits on-chain and earning rewards for providing practical value to the community.

### ✨ Key Features
* **On-Chain Check-in:** Every location visit creates a verifiable proof on the Base network.
* **NCAT Reward System:** Automatic token distribution for users who suggest spots and check-in.
* **Curated Spots:** Locations personally experienced and approved by the nomad community.
* **Global Nomad Network:** Share hidden gems, from remote campsites to coworking spaces.

---

## 🛠️ Technical Details
* **Framework:** React / Vite (TypeScript)
* **Blockchain:** Base Network (L2)
* **Token:** NCAT (NomadCat)
* **Styling:** Tailwind CSS

---

## ⚙️ Installation (Local Setup)

1. **Install Dependencies:** `npm install`
2. **Environment Variables:** Copy `.env.example` to `.env` and add your API keys.
3. **Run the App:** `npm run dev`
